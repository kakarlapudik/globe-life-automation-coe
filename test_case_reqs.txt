Use Case 1: Verify Globe Life Investor Relations Homepage Content Links
Description: Launch the Globe Life investor relations website and verify only homepage-specific content links (main content area, hero section, featured content) are working and return HTTP status code 200
URL: https://investors.globelifeinsurance.com/

Preconditions:
- Internet connection is available
- Website is accessible
- Browser is installed

Steps:
1. Navigate to https://investors.globelifeinsurance.com/
2. Wait for page to load completely
3. Extract ONLY homepage-specific links from main content areas using selectors: main a[href], .hero a[href], .content a[href], .homepage a[href], .banner a[href], .featured a[href]
4. Exclude navigation menu links and footer links (tested separately)
5. If no specific homepage links found, use main content area links only
6. Limit to first 10 links to focus on homepage content
7. For each homepage link, extract href attribute and validate URL
8. Send HTTP HEAD request to verify status code using requests library
9. Log each link validation result with URL and status code
10. Save detailed JSON report with homepage-specific link validation results
11. Assert that no broken homepage content links are found

Expected Results:
- Homepage loads successfully
- Only homepage content links are extracted (not navigation/footer)
- All homepage content links return HTTP status code 200
- No broken homepage links are found
- Validation report focuses on homepage content only
- HTML report includes list of homepage-specific validated links with status codes
- HTML report shows homepage link validation summary with pass/fail counts

Priority: High
