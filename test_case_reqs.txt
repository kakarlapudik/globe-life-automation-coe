Use Case 1: Verify Globe Life Investor Relations Homepage Links
Description: Launch the Globe Life investor relations website and verify all links on the home page are working and return HTTP status code 200
URL: https://investors.globelifeinsurance.com/

Preconditions:
- Internet connection is available
- Website is accessible
- Browser is installed

Steps:
1. Navigate to https://investors.globelifeinsurance.com/
2. Wait for page to load completely
3. Extract all links (anchor tags) from the homepage using page.locator("a[href]").all()
4. For each link, extract href attribute and validate URL
5. Send HTTP HEAD request to verify status code using requests library
6. Log each link validation result with URL and status code
7. Print comprehensive summary showing total links, broken links count
8. Save detailed JSON report with all link validation results
9. Assert that no broken links are found (except acceptable 403 status codes)

Expected Results:
- Homepage loads successfully
- All links are extracted correctly
- All links return HTTP status code 200
- No broken links are found
- Validation report is generated
- HTML report includes complete list of all validated links with status codes
- HTML report shows link validation summary with pass/fail counts

Priority: High
